상세설명

서버환경 선택 가능
  • PHP 7.4 / mariadb-10.0.x / UTF-8 / HTTP2.0 / Apache2 / InnoDB 지원
  • PHP 7.3 / mariadb-10.0.x / UTF-8 / HTTP2.0 / Apache2 / InnoDB 지원
  • PHP 7.0 / mariadb-10.0.x / UTF-8 / HTTP2.0 / Apache2 / InnoDB 지원
  • PHP 5.5 / Mysql 5.x / UTF-8
  • PHP 5.5 / Mysql 5.x / EUC-KR
  • 워드프레스, 그누보드 등 프로그램 자동설치 및 MySQLi 지원
프로그램 자동설치
  • UTF-8 환경에서
    워드프레스, 제로보드XE, 그누보드, 킴스큐, 텍스트큐브 자동설치를 제공합니다.
  • PHP7 환경에서
    킴스큐, 텍스트큐브, XE 등 일부 프로그램은 자동설치가 지원되지 않습니다.
SSL 향상
  • SNI 지원
  • SSL 기본포트(443) 사용가능
Language / Script
  • cgi, perl script, gcc, C/C++ Complier, SSI, Flash, DBI/DBD, zend optimizer, Python
  • GD라이브러리를 제외한 다른 lib나 파일을 설치해서 사용할 시에는 작동을 안하거나 비정상적으로 작동할 수 있습니다.
  • php7 서버 환경에서는 cgi 사용이 불가능합니다.
FTP/SFTP/SSH 지원
  • SSH 터널링(Tunneling)은 지원하지 않습니다.
DB 환경
  • MySQL은 Localhost로 기본 접속되며 설정은 신청하면 원격으로 설정됩니다.
이메일 계정
  • 각 계정의 메일 용량은 5MB
  • [계정 수]
    기본형- 3개, 일반형- 10개,
    [고급형, 프리미엄]- 30개, 프리미엄 플러스- 50개
  • 메일 발송은 한번에 30통/ 하루에 500통까지만 발송이 가능합니다.
  • 자정(00시)에 초기화 됩니다.
월사용료 혜택
  • 신청하시는 서비스 기간에 따라 월사용료 할인 혜택을 제공합니다.
  • 6개월(5%), 12개월(10%), 24개월(12%), 36개월(15%)
서비스 제한
  • 서버 부하가 과도한 계정의 경우 서비스가 취소될 수 있습니다.
  • DB용량은 서버 디스크 한도 내에서 무제한으로 제공되며 과도한 이용 시 제한될 수 있습니다.
  • 무료 제공되는 스트리밍 서비스는 트래픽 폭주 시 일시적으로 서비스를 제한할 수 있습니다.
서비스 변경 / 추가
  • 서비스를 사용하다 용량이 부족한 경우, 상위 서비스로의 변경이 가능합니다.
  • (하위 서비스로의 변경은 불가능합니다.)
